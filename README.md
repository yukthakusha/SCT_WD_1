# Responsive Landing Page

This project is a responsive landing page created as **Task 1** for the **SkillCraft Technology Web Development Internship**.  
It features a fixed and interactive navigation menu with hover and scroll-based style changes.

## Features
- Fixed navigation bar visible across the page
- Navbar style changes on scroll
- Hover effects on navigation menu items
- Fully responsive design for mobile, tablet, and desktop screens
- Clean and modern UI layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Purpose
The goal of this task is to demonstrate:
- Responsive web design principles  
- Interactive UI behavior using CSS and JavaScript  
- Proper project structure and GitHub usage
